## Environment

Create a `.env` file with all needed params specified in `.env.example`

## Install and run

- Run `composer install`
- Run `php artisan key:generate`
- Run `php artisan migrate`
- Run `php artisan db:seed` to run seeders.
- Run `php artisan serve`

## Install libraries

- Run `npm install`
- Run `npm run build`
